# Orquestrador — skill Claude Code

Leia `ORQUESTRADOR.md` na raiz do repositório e aplique o fluxo de decisão definido ali.

Passos:
1. Leia `ORQUESTRADOR.md`.
2. Classifique o pedido do usuário (§ 2 do orquestrador).
3. Selecione o agente correto (§ 3).
4. Monte o contexto mínimo (§ 4).
5. Verifique coordenação com Codex (§ 5).
6. Execute usando a skill correspondente ou informe ao usuário qual agente deve ser acionado.
7. Registre o resultado (§ 7).
