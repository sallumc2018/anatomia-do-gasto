# Planejamento de Campanhas de Recrutamento de Voluntários

Este documento compila as estratégias, rascunhos de comunicação e guias para a atração de voluntários cívicos e colaboradores de código aberto para a plataforma **Anatomia do Gasto**.

---

## 1. Sequência de Postagens para o LinkedIn

Abaixo está o cronograma de postagens planejadas para divulgar a plataforma e atrair perfis técnicos e de comunicação:

### Post 1: Apresentação do Propósito (Narrativa Cívica)
> **Título:** Onde o dinheiro público é realmente aplicado?
> 
> A transparência das contas públicas não deve exigir um diploma em contabilidade governamental. Desenvolvi o **Anatomia do Gasto**, uma plataforma cívica independente e de código aberto que traduz relatórios orçamentários complexos em linguagem cidadã e gráficos auditáveis.
>
> Iniciei o piloto por Sorocaba/SP, abrindo os dados de Saúde e Educação (2020-2025). Toda a metodologia de extração e o código-fonte estão disponíveis no GitHub para qualquer cidadão auditar.
>
> Quero expandir e preciso de ajuda para decifrar mais municípios. Se você acredita no poder da tecnologia cívica e do controle social, venha colaborar com o projeto.
>
> 🔗 Conheça o projeto e saiba como ajudar: https://www.anatomiadogasto.ong.br/voluntarios
>
> #TecnologiaCivica #OpenSource #DadosAbertos #Transparencia

### Post 2: Foco em Perfis Técnicos (Next.js & Python)
> **Título:** Procurando seu próximo projeto de código aberto com impacto real?
>
> O **Anatomia do Gasto** foi desenvolvido sob uma arquitetura moderna e escalável:
> - **Frontend:** Next.js, React, TypeScript e TailwindCSS.
> - **Pipelines de dados:** Python 3.12, coletando dados de portais oficiais (SICONFI, SIOPS, PNCP).
>
> O repositório público no GitHub está estruturado com guias claros de contribuição (`CONTRIBUTING.md`) e tarefas fáceis de configuração para quem deseja colaborar.
> 
> Se você quer aplicar seus conhecimentos em Next.js ou Python para fortalecer o controle social no Brasil, existem tarefas abertas aguardando sua contribuição.
>
> 🔗 Acesse o GitHub do projeto e veja as tarefas marcadas como "good first issue": https://github.com/sallumc2018/anatomia-do-gasto
>
> #NextJS #Python #Programacao #Github #DesenvolvimentoWeb

### Post 3: Foco em Perfis de Dados e Comunicação
> **Título:** Transparência exige mais do que apenas código
>
> Para que os dados públicos façam sentido, o projeto precisa de analistas, jornalistas e designers que saibam traduzir números frios em narrativas compreensíveis.
> 
> No **Anatomia do Gasto**, colaboradores atuam em:
> - **Auditoria Cívica:** Validando se os dados do site condizem com os portais oficiais locais.
> - **Escrita Cívica:** Redigindo glossários e explicações simples sobre termos orçamentários.
> - **Design & UI:** Tornando a visualização dos fluxos financeiros intuitiva.
>
> Não é preciso saber programar para contribuir com a transparência da administração pública.
>
> 🔗 Saiba mais na página de colaboradores do projeto: https://www.anatomiadogasto.ong.br/voluntarios
>
> #JornalismoDeDados #DesignDeInteracao #CienciaDeDados #ControleSocial

## 2. Guia de Integração em Comunidades Especializadas de Dados

### A. Comunidade Brasil.IO
*   **O que é:** Repositório colaborativo de dados públicos mantido por voluntários.
*   **Como acessar:** Acesse o convite para o **Discord** ou **Telegram** diretamente no site oficial (https://brasil.io).
*   **Como interagir (Etiqueta):**
    1. Apresente-se no canal `#apresentacao` ou `#geral` com foco profissional e individual: "Olá, sou o Alexandre, moro em São Paulo/Capital e sou o idealizador do Anatomia do Gasto..."
    2. Apresente a plataforma no canal `#projetos` ou `#colaboracao` de forma direta: "Desenvolvi o Anatomia do Gasto, um projeto Next.js/Python de código aberto que lê dados do SICONFI/SIOPS e traduz em gráficos simples para o cidadão comum. O projeto é open-source e está buscando desenvolvedores e analistas de dados. Link do GitHub: github..."

### B. Fórum da Open Knowledge Brasil (OKBR)
*   **O que é:** Rede nacional de tecnologia cívica e dados abertos.
*   **Como acessar:** Cadastre-se no Fórum de Discussão da OKBR (https://discuss.okfn.org/) ou nos canais da comunidade do projeto *Querido Diário*.
*   **Como interagir:** Crie um tópico explicativo na categoria de projetos cívicos:
    *   **Título:** Apresentação de Projeto: Anatomia do Gasto — Orçamentos Municipais em Linguagem Cidadã.
    *   **Conteúdo:** Apresente de forma clara a stack tecnológica, a metodologia do piloto em Sorocaba/SP e convide a comunidade de dados abertos a auditar o projeto e enviar sugestões de melhorias.

## 3. Adiamento de Campanhas com Universidades

> [!IMPORTANT]
> **Status de Parcerias Acadêmicas:** Adiado.
> 
> **Análise de Viabilidade:** O estabelecimento de convênios formais de extensão universitária ou a validação garantida de horas complementares exige, obrigatoriamente, uma personalidade jurídica constituída com inscrição ativa no Cadastro Nacional da Pessoa Jurídica (CNPJ). 
> 
> Como o projeto opera atualmente de forma informal no modelo de colaboração de código aberto (Caminho A), sem CNPJ constituído, a abordagem ativa a coordenações universitárias foi postergada até que a formalização jurídica da associação seja realizada.
