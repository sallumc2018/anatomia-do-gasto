# Auditoria do Anatomia do Gasto

A auditoria cobre a cadeia completa de remuneração e destinação de recursos públicos dos agentes políticos que atuam em Sorocaba. O escopo inclui vereadores, prefeito, vice, deputados estaduais e federais com atuação no município.

## Princípios

- **Rastreamento total**: toda entrada (salários, verbas, cotas) e toda saída (emendas, destinações) deve ser identificada e mapeada.
- **Armazenamento eterno**: os dados são coletados continuamente, retroagindo ao passado e seguindo para o futuro, para permitir comparações históricas.
- **Ranking de ações**: cada parlamentar terá um perfil com indicadores positivos e negativos baseados em seu histórico.
- **Transparência radical**: quando não for possível rastrear algo, o motivo será explicitado.
