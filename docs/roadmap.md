# Roadmap do Anatomia do Gasto

Este documento descreve a visão de evolução do projeto em fases. As datas são estimativas e podem mudar conforme a disponibilidade do mantenedor e a chegada de colaboradores.

---

## Fase 1: Fundação (Maio–Julho 2026)

**Objetivo:** Pipeline de dados sólido para saúde e educação em Sorocaba.

- [ ] Verificar integridade dos dados de saúde 2020–2023.
- [ ] Executar verificação de integridade para educação 2024–2025.
- [ ] Implementar script de coleta de remuneração e emendas de agentes políticos (trilha de auditoria).
- [ ] Criar visualizações de auditoria: tabelas, grafos, rankings.
- [ ] Publicar 1 "Data Poke" por mês no Twitter/X.
- [ ] Contato formal com UFSCar Sorocaba para Câmara 2.

---

## Fase 2: Validação (Agosto–Dezembro 2026)

**Objetivo:** Gerar autoridade técnica e as primeiras denúncias.

- [ ] Expandir pipeline para transporte e segurança pública.
- [ ] Recrutar 3–5 moderadores voluntários para Câmara 2 (estudantes de direito UFSCar).
- [ ] Implementar MVP da Câmara 3 (sistema de comentários com login gov.br).
- [ ] Parceria formal com 1 veículo de jornalismo local.
- [ ] Publicar relatório trimestral de achados.
- [ ] Botão de denúncia formatada para Fala.BR.

---

## Fase 3: Expansão (2027)

**Objetivo:** Sair de Sorocaba e ganhar escala.

- [ ] Adicionar 3 novos municípios (pipeline modular).
- [ ] Implementar agente de IA para narrativas automáticas (LLM traduzindo dados para linguagem cidadã).
- [ ] Radar de gastos dos representantes — MVP para vereadores de Sorocaba.
- [ ] Programa de Embaixadores com 10+ cidadãos ativos.
- [ ] Buscar financiamento via editais (Fundação Lemann, Open Society, CGI.br).

---

## Fase 4: Sistêmica (2028+)

**Objetivo:** Impacto nacional e mudança estrutural.

- [ ] Cobertura de todos os municípios de SP.
- [ ] Integração com TCU e TCEs (dados estruturados como insumo oficial).
- [ ] Advocacy legislativo (propostas de melhoria da transparência ativa).
- [ ] Banco de dados em grafo (Neo4j) para cruzamento de vínculos.
- [ ] Sustentabilidade financeira (doações + editais + serviços).

---
